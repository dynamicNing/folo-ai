# ADR-003: 管理后台使用 JWT 鉴权，单管理员模式

- **日期**：2026-04-16
- **状态**：已接受

## 背景

管理后台需要鉴权，防止内容被公开篡改或删除。站点为个人使用，不需要多用户体系。

## 决策

使用 JWT（JSON Web Token）做无状态鉴权。密码以 bcrypt hash 存储在 `.env` 环境变量中，不建数据库用户表。Token 有效期 7 天，存储在浏览器 localStorage。

## 原因

- 单管理员场景不需要用户表，`.env` 存 hash 足够
- JWT 无状态，服务重启不丢失登录态
- 实现简单，维护��本低

## 后果

**正面**：无需数据库；部署时只需配置两个环境变量。  
**负面**：Token 无法主动吊销（只能等过期）；密码修改需重新生成 hash 并重启服务。  
**中性**：如未来需要多用户或细粒度权限，需引入用户表和 refresh token 机制。
