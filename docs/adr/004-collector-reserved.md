# ADR-004: AI 采集模块预留接口，暂不实现

- **日期**：2026-04-16
- **状态**：已接受

## 背景

站点的核心价值之一是展示 AI 采集的信息。但采集逻辑依赖外部数据源和 AI 服务，当前阶段优先跑通基础展示和管理流程。

## 决策

在 `src/services/collector.js` 中预留模块接口（`collect`、`getConfig`、`saveConfig`），所有方法抛出 `Not implemented` 错误。后端路由 `POST /api/collector/run` 等返回 HTTP 501。前端 `/admin/collector` 页面显示占位说明。

## 原因

- 避免过早设计未验证的采集逻辑
- 接口预留让未来实现不需要改动路由和前端入口
- 当前手动维护 MD 文件可以满足内容需求

## 后果

**正面**：系统保持简单；未来实现采集只需填充 `collector.js` 内部逻辑，外部接口不变。  
**负面**：内容更新依赖手动写 MD 文件，效率有限。  
**中性**：实现时需决定采集结果的 MD 文件命名规范和 frontmatter 字段映射。
